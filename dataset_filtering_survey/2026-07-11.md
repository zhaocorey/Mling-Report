## 📝 新增博客/教程

### 1. Data for Agents（NVIDIA / HuggingFace 联合博客）

- **发布日期**: 2026-07-08
- **来源**: NVIDIA & HuggingFace
- **核心主题**: 面向 AI Agent 的开放数据策划策略

#### 内容摘要

该博客讨论了三个与 data curation 高度相关的主题：

1. **Nemotron Post-Training v3 Prompt Atlas** — 一个交互式数据混合可视化工具，允许研究者直观探索不同数据混合比例对模型能力的影响。这是数据混合优化从「数值搜索」走向「人机交互」的重要一步。

2. **面向 Agent 的开放数据策划策略** — 讨论了为 AI Agent 训练构建高质量数据集的挑战，包括：
   - Agent 任务数据的多样性要求（规划、工具使用、多步推理）
   - 合成数据 vs 真实数据的权衡
   - 数据质量评估的 Agent 特化需求

3. **多语言 Agent 的数据质量考量** — 涉及多语言场景下的特殊挑战：
   - 不同语言的工具调用数据质量差异
   - 跨语言 Agent 能力的评估与数据覆盖
   - 低资源语言 Agent 数据的构建策略

#### 实用价值

⭐⭐⭐⭐ — **Agent 数据策划是 data curation 的新兴子领域，与我们的多语言数据集构建有直接交叉。**

- (1) **Prompt Atlas 可视化工具**：交互式混合比例探索的思路可迁移到多语言混合比例的可视化分析，帮助我们直观理解不同语言配比对模型能力的影响
- (2) **Agent 数据质量标准**：随着 Agent 训练需求的增长，理解 Agent 数据的质量维度有助于扩展我们的数据评估框架
- (3) **多语言 Agent 数据**：博客中提到的多语言质量考量与我们 6/30 报告中 MuRating 的跨语言质量迁移方法形成互补

### 二、Data Curation 向 Agent 方向延伸

「Data for Agents」博客标志着一个重要趋势：**数据策划正在从「通用 LLM 训练」向「特定能力训练」延伸**。

| 阶段 | 数据策划焦点 | 代表工作 |
|------|-------------|---------|
| 2024-2025 | 通用预训练数据 | FineWeb, DataComp-LM, SmolLM |
| 2026 H1 | Post-training / 对齐数据 | CuratorKIT, BeeS, Alignment Pipeline |
| 2026 H2（趋势） | **Agent 特化数据** | Data for Agents, Prompt Atlas |

这与基线中已有的趋势一致：
- LHAS（6/30）关注多阶段训练的长期数据选择效应
- Alignment Pipeline Survey（6/30）将对齐数据视为显式 pipeline
- Agent 数据策划是这一方向的自然延伸

### 三、交互式数据混合分析兴起

Prompt Atlas 的出现与 DeMix（6/30 新增）形成了有趣对比：

| 方法 | 技术路线 | 交互方式 |
|------|---------|---------|
| DeMix | 模型合并预测最优混合比 | 自动化搜索 |
| Prompt Atlas | 交互式可视化探索 | 人机交互 |

两种方法互补：DeMix 提供数学最优解，Prompt Atlas 提供人类直觉验证。在多语言混合场景中，结合两者可能是最佳策略——先用 DeMix 搜索数学最优比例，再用可视化工具验证和调整。
