## 🆕 新增论文

### 1. [CuratorKIT: Data Curation and Synthetic Data Generation for LLM Post-Training](https://arxiv.org/abs/2606.21631)

- **作者**: 详见论文 | **年份**: 2026-06
- **方法摘要**: 开源 Python 库，覆盖后训练数据管理全生命周期。支持 6 种数据格式读取（含自动 schema 检测）、预生成过滤、合成数据生成、质量过滤和溯源追踪，全部封装在单一可配置 pipeline 中。既是一篇方法论文，也是一个即插即用的工程工具。
- **关键方法**: `post-training curation` `unified pipeline` `synthetic data generation` `provenance tracking` `multi-format ingestion`
- **实用价值**: ⭐⭐⭐⭐⭐ — **工具+论文双重价值**。理由：(1) 直接解决后训练数据策划的工程碎片化问题，不需要拼凑多个工具；(2) 溯源追踪功能在多语言场景中尤为重要——需要追踪每条数据来自哪个语言、哪个源头；(3) 开源意味着可以立即集成到我们的 pipeline 中。**建议精读其 pipeline 架构设计，评估与 DataTrove 的互补性。**

### 2. [Provenance-Grounded Gating and Adaptive Recovery in Synthetic Post-Training Data Curation](https://arxiv.org/abs/2606.11127)

- **作者**: 详见论文 | **年份**: 2026-06
- **方法摘要**: 对合成后训练数据过滤中两个被忽视的实践进行受控研究：(1) 过滤信号是否应基于生成该样本的源证据（provenance grounding）；(2) 被过滤拒绝的样本是否可以自适应恢复。核心发现：幻觉门控（hallucination gating）和奖励门控（reward gating）拒绝的样本几乎不重叠，说明多维度门控策略的必要性。
- **关键方法**: `provenance-grounded filtering` `adaptive recovery` `multi-dimensional gating` `synthetic data quality` `hallucination gating`
- **实用价值**: ⭐⭐⭐⭐ — 对多语言合成 SFT 数据的质量保障有直接参考价值。在多语言合成中，不同语言的"幻觉模式"不同，需要语言感知的溯源门控。**建议阅读 Section 4（门控策略对比实验），评估多维度门控在多语言场景的适用性。**

### 3. [Data Value Density Enhancement for Large Language Model Training: A Comprehensive Survey](https://papers.ssrn.com/sol3/papers.cfm?abstract_id=6618802)

- **作者**: 详见论文 | **年份**: 2026
- **方法摘要**: 提出"数据价值密度提升"（DVD Enhancement）的统一框架，涵盖通过过滤、采样、合成、混合等方法最大化训练数据价值的方法论。这是首个试图将碎片化的数据策划方法统一到单一理论框架下的综述工作。
- **关键方法**: `data value density` `unified framework` `survey` `filtering + sampling + synthesis`
- **实用价值**: ⭐⭐⭐ — 综述性质，理论框架可能有启发价值，但不直接产出工具或方法。**建议按需查阅，关注其 DVD 度量指标的定义是否具有可操作性。**

### 4. [DataRater: Meta-learned Dataset Curation](https://proceedings.neurips.cc/paper_files/paper/2025/hash/1cfb3b066461f1c542be5242a622ce4b-Abstract-Conference.html)

- **作者**: 详见论文 | **年份**: 2025 (NeurIPS)
- **方法摘要**: 通过元学习自动学习"什么数据对训练有价值"，而非依赖人工设计的质量启发式。可扩展、细粒度的数据策划，通过学习而非规则来确定数据质量。
- **关键方法**: `meta-learning` `learned data quality` `automated curation`
- **实用价值**: ⭐⭐⭐⭐⭐ — **NeurIPS 2025 论文，元学习数据策划的重要工作**。学习的 curation 策略可能比 propella-1 的标注维度更自适应。**建议精读，评估其元学习框架是否支持多语言数据场景。**

### 5. [ASLEC: On the Step Length Confounding in LLM Reasoning Data Selection](https://aclanthology.org/2026.findings-acl.918/)

- **作者**: 详见论文 | **年份**: 2026 (ACL Findings)
- **方法摘要**: 发现 LLM 推理数据选择中的"步长混淆"问题：基于 log-probability 的数据评分系统性地偏好更长的推理步骤，而非更高质量的推理。提出 ASLEC 方法来纠正这一偏差。
- **关键方法**: `step length confounding` `reasoning data selection` `bias correction`
- **实用价值**: ⭐⭐⭐⭐ — ACL 2026 Findings，对推理数据选择有重要警示意义。多语言推理数据中，不同语言的"自然推理步长"不同，步长混淆问题可能更严重。

### 6. [InstructDiff: Domain-Adaptive Data Selection via Differential Entropy](https://arxiv.org/abs/2601.23006)

- **作者**: 详见论文 | **年份**: 2026-01
- **方法摘要**: 利用基础模型和最小指令微调模型之间的微分熵作为域自适应选择标准。推理任务偏好熵增加（认知扩展），通用任务偏好熵减少（对齐压缩）。
- **关键方法**: `differential entropy` `domain-adaptive selection` `instruction tuning`
- **实用价值**: ⭐⭐⭐⭐ — 域自适应选择思路对多语言场景有迁移价值。不同语言可视为不同"域"，InstructDiff 提供了一种自动化的域感知选择方法。

### 7. [20/20 Vision Language Models: Better VLMs through Data Curation Alone](https://arxiv.org/abs/2605.11405)

- **作者**: 详见论文 | **年份**: 2026-05
- **方法摘要**: 证明仅通过数据策划（固定架构、训练和计算量）就能将 VLM 性能提升 +11.7pp（跨 20 个基准）。策划后的 2B 模型超越了 17 倍计算量的 InternVL3.5-2B。
- **关键方法**: `data curation only` `VLM` `compute-efficient`
- **实用价值**: ⭐⭐⭐ — 聚焦 VLM 而非纯 LLM，但"仅靠数据策划提升 11.7pp"的结论有力地支持了数据策划投资的 ROI。
