### 1. Co-Adaptive Multi-Task LoRA: Transfer-Aware, Label-Free Control of Domain Participation

| 字段 | 内容 |
|------|------|
| **作者** | Wei Zhang, Lin Tang, Ming Zhao, Yuxuan Wang |
| **年份** | 2026 |
| **链接** | https://arxiv.org/abs/2607.03522 |
| **类别** | Mixing / Domain Participation |

**方法摘要**：提出 CoDA（Co-Adaptive）控制器，通过对共享 LoRA adapter 在小规模无标注 probe 上的前向传播，获取每个 domain 的能力信号（剩余提升空间和学习速度）以及跨 domain 的迁移亲和度。将这些信号组合为熵正则化二次规划问题，在 simplex 上求解每个 domain 的参与度（包括 loss 权重和数据采样份额）。前向传播仅计算，不引入额外可训练参数。

**关键方法**：
- 无标注 probe 上的能力信号和迁移亲和度估计
- 熵正则化 QP 求解 domain 参与度
- 前向传播即可，无需梯度计算

**实用价值评估**：⭐⭐⭐⭐（重要参考）
- 直接解决多 domain LoRA 训练中的 domain 权重和采样比例问题
- 无需标注数据即可自动调整 domain 参与
- 对多语言数据集构建中的 domain 混合策略有直接参考价值

### 2. CuBAS: Information Geometric Curvature-Based Adaptive Sampling for Supervised Classification

| 字段 | 内容 |
|------|------|
| **作者** | Alexandre L. M. Levada |
| **年份** | 2026 |
| **链接** | https://arxiv.org/abs/2607.03145 |
| **类别** | Sampling / Active Learning |

**方法摘要**：基于信息几何的自适应数据选择框架。将标注数据集视为统计流形，使用 q-state Potts 马尔可夫随机场模型，通过二阶与一阶 Fisher 信息的比值估计局部曲率。构建 k-NN 图并在每个顶点推导闭式曲率分数，将数据划分为低曲率（平滑同质簇）和高曲率（决策边界附近）两个互补区域，从两类区域中选择样本构建紧凑且信息量最大的训练子集。

**关键方法**：
- Potts MRF 建模标注数据的统计流形
- 闭式曲率分数计算（线性复杂度）
- 双区域（低曲率 + 高曲率）采样策略

**实用价值评估**：⭐⭐⭐（参考）
- 提供基于微分几何的理论视角理解数据选择
- 计算效率高（线性复杂度），适合大规模应用
- 主要面向分类任务，对 LLM 预训练的直接适用性有限，但理论框架有启发价值

### 3. Power and Limits of Subset Selection in Statistical Estimation

| 字段 | 内容 |
|------|------|
| **作者** | Dina Barak-Pelleg, Daniel Berend |
| **年份** | 2026 |
| **链接** | https://arxiv.org/abs/2607.03391 |
| **类别** | Theory / Data Selection |

**方法摘要**：通过"超级教学"（super-teaching）框架研究子集选择在统计估计中的能力和局限。证明对于均值估计，任何在均值邻域内密度有下界的分布都可以实现超级教学，允许子集规模 k = o(n^{1/3})。将分析扩展到方差的平滑泛函等参数，包括重尾分布场景。还证明超级教学可以显著改善样本中位数等非线性估计量的收敛速率，甚至在 MLE 不一致或渐近正态性失效的场景下仍然有效。

**关键方法**：
- Super-teaching 理论框架
- 子集选择的收敛速率分析
- 非线性估计量的改进理论

**实用价值评估**：⭐⭐⭐（参考）
- 为数据选择的有效性提供理论基础
- 统一理论框架可指导 LLM 训练数据子集选择的理论分析
- 纯理论工作，需要进一步实验验证其在 LLM 场景的适用性

### 4. QuaSR: Quality-Aware Sample Reweighting for Pacific Indigenous Speech Recognition

| 字段 | 内容 |
|------|------|
| **作者** | Yishun Li, Yang Xiao, Gongping Huang, Eun-Jung Holden, Nick Thieberger, Ting Dang |
| **年份** | 2026 |
| **链接** | https://arxiv.org/abs/2607.03658 |
| **类别** | Quality / Sampling |

**方法摘要**：面向低资源语言的 ASR 自适应，提出质量感知样本重加权框架。结合数据侧可靠性（声学质量、转录一致性、对齐可靠性的综合评估）和模型侧可学习性（基于训练 loss 的样本难度），产生统一的样本效用分数作为训练权重。在四种太平洋原住民语言上评估，证明质量-可学习性双信号效用分数与自适应性能可靠相关，并持续优于标准微调和替代数据选择策略。

**关键方法**：
- 数据可靠性 + 模型可学习性双信号融合
- 样本效用分数驱动的重加权
- 低资源语言 ASR 适配

**实用价值评估**：⭐⭐⭐⭐（重要参考）
- **直接涉及低资源语言**，对多语言数据集构建有重要参考价值
- 双信号（数据质量 + 模型可学习性）融合思路可迁移到 LLM 数据选择
- 为质量过滤提供"不仅看数据本身，还要看模型能否学到"的互补视角
